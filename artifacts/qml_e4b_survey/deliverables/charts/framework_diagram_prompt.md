# Framework Diagram Prompt

**Paper**: Quantum Advantage in Machine Learning: A Systematic Taxonomy of Computational Bottlenecks

## Image Generation Prompt

A sophisticated, vector-art methodology framework architecture diagram illustrating a multi-stage data flow, suitable for a top-tier machine learning conference paper. The overall flow should proceed from left to right, set against a pristine white background. Use a muted, academic color palette: primary data flow elements in deep slate blue (#4477AA), secondary processing nodes in teal (#44AA99), accent highlights using muted gold/yellow (#CCBB44), and structural annotations in soft purple (#AA3377). The style must be clean, flat design with subtle, academic dimensionality (e.g., very soft inner shadows or gradients, avoiding photorealism).

The process must be clearly segmented into four distinct, rounded-corner modules: 1. Initial Input Sources (Sources: arXiv, Major Conferences, Journals), connected by thin arrows. 2. Core Processing Stage (Module: Iterative Keyword Query & Filtering), containing sub-steps like "Keyword Combination" and "Inclusion/Exclusion Criteria Check," using the teal color. 3. Expert Review & Curation (Module: Foundational Article Selection), emphasizing rigorous filtering. 4. Output/Synthesis (Final Module: Taxonomy Generation).

Use directional arrows (slate blue) to guide the flow sequentially, showing the refinement process. Label the key transitions or decision points with minimal text annotations, such as "Initial Pool (>300 Docs)" leading to a decision diamond labeled "Theoretical Rigor Check," which splits the flow toward the final category structure. The final Taxonomy Generation module should visually represent the emergent structure by containing three smaller, interconnected concept nodes (e.g., "Feature Space Expansion," "Hamiltonian Evolution," "Quantum Kernel Analysis"), connected by purple lines, indicating the structural underpinning source. The overall composition must feel highly systematic, precise, and complex, yet visually breathable.

## Usage Instructions

1. Copy the prompt above into an AI image generator (DALL-E 3, Midjourney, Ideogram, etc.)
2. Generate the image at high resolution (2048x1024 or similar landscape)
3. Save as `framework_diagram.png` in the same `charts/` folder
4. Insert into the paper's Method section using:
   - LaTeX: `\includegraphics[width=\textwidth]{charts/framework_diagram.png}`
   - Markdown: `![Framework Overview](charts/framework_diagram.png)`
