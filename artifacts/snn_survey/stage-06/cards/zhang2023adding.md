# Adding Conditional Control to Text-to-Image Diffusion Models

*source: s2_fulltext | tokens: 9109*

This paper introduces ControlNet, a neural network architecture designed to add fine-grained spatial conditioning controls to large, pretrained text-to-image diffusion models, specifically Stable Diffusion. The core problem addressed is the lack of precise spatial control in standard diffusion models, where users struggle to dictate layouts, poses, or shapes via text prompts alone without trial-and-error. ControlNet solves this by enabling end-to-end learning of diverse conditional controls (e.g., edges, depth, pose, segmentation) without compromising the quality of the backbone model.

The technical approach involves creating a trainable copy of the pretrained diffusion model's encoding layers while keeping the original parameters frozen. These two branches are connected via "zero convolution" layers (1×1 convolutions initialized to zero weights and biases). This initialization ensures that at the start of training, the trainable copy does not introduce harmful noise into the backbone, allowing the pretrained features to remain robust. The architecture supports various conditioning inputs processed through a lightweight encoder. Training is robust across dataset sizes, demonstrated on small (<50k) and large (>1m) datasets.

Quantitative results validate the method's efficacy. ControlNet trained on 200k depth samples using a single RTX 3090Ti achieved results indistinguishable from the industrial SDv2-D2I model, which required a large A100 cluster and 12M images. User studies involving 12 participants and 100 rankings showed ControlNet outperformed baselines like PITI and Sketch-Guided Diffusion in both image quality and condition fidelity. On the ADE20K segmentation dataset, ControlNet achieved a reconstructed Intersection-over-Union (IoU) comparable to state-of-the-art segmentation methods, with strong FID and CLIP aesthetic scores. The method also supports composing multiple conditions (e.g., edges and pose simultaneously) without linear interpolation.

ControlNet relates to prior work on finetuning and adapters (e.g., LoRA, T2I-Adapter) by offering a stable backbone reuse strategy. While concurrent work like T2I-Adapter exists, ControlNet's zero-convolution design offers distinct training stability. Limitations include the need for specific training data for each condition and the computational overhead of the trainable copy (23% more GPU memory, 34% more time per iteration). For a survey on Set Neural Networks, this work is relevant regarding the handling of multiple conditional inputs as a set of constraints, though the architecture is primarily a U-Net based diffusion controller rather than a permutation-invariant set network.
