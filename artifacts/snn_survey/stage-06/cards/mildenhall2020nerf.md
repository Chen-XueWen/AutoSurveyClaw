# NeRF: Representing Scenes as Neural Radiance Fields for View Synthesis

*source: arxiv_pdf | tokens: 12821*

Mildenhall et al. (2020) introduce Neural Radiance Fields (NeRF), a method for high-fidelity novel view synthesis from sparse input views. The approach represents a static scene as a continuous 5D function parameterized by a fully-connected deep network (MLP). The network maps a single continuous 5D coordinate—comprising spatial location (x, y, z) and viewing direction (θ, φ)—to volume density and view-dependent emitted radiance. This coordinate-based representation allows the model to encode complex geometry and appearance without the storage costs of discretized voxel grids. To enable effective optimization, the authors incorporate positional encoding (Fourier features) to map inputs into a higher-dimensional space, overcoming spectral bias and allowing the MLP to represent high-frequency scene content. Additionally, a hierarchical sampling procedure allocates samples proportionally to expected scene content, optimizing the use of network capacity.

The method is optimized end-to-end using differentiable volume rendering, minimizing the error between rendered and observed RGB images. Architectural details include an 8-layer MLP with 256 channels per layer and skip connections, followed by a view-dependent head. Ablation studies demonstrate that positional encoding and view-dependent color prediction contribute most significantly to performance, with the full model outperforming variants lacking these components.

Quantitative evaluation on the DeepVoxels dataset yields a PSNR of 40.15 and SSIM of 0.991, surpassing Local Light Field Fusion (LLFF) (PSNR 34.38) and Scene Representation Networks (SRN) (PSNR 33.20). On a custom realistic synthetic dataset featuring non-Lambertian materials, NeRF achieves 31.01 PSNR compared to 26.05 for Neural Volumes (NV). On real forward-facing scenes (LLFF benchmark), it produces superior multiview consistency and fewer artifacts than LLFF, despite using only the input images for training.

Key limitations include slow training (100–300k iterations, ~1–2 days on a V100) and inference speed (~30 seconds per frame). However, storage is highly efficient (5 MB weights vs. LLFF's >15 GB). For a survey on Set Neural Networks, NeRF is critical as it exemplifies coordinate-based implicit representations. It processes rays as sets of query points, where the MLP acts as a permutation-invariant function approximator over continuous coordinates, bridging the gap between neural set processing and continuous scene modeling.
