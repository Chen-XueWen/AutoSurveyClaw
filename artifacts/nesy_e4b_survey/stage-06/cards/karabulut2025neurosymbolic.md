# Neurosymbolic Association Rule Mining from Tabular Data

*source: arxiv_html | tokens: 10225*

$\text{Aerial+}$ presents a novel neurosymbolic approach to address the critical challenge of "rule explosion" in Association Rule Mining (ARM) when applied to high-dimensional tabular data. The core problem is that traditional exhaustive ARM algorithms generate an excessive number of rules, leading to prohibitive execution times, which consequently degrades the performance of downstream rule-based interpretable Machine Learning models.

The technical approach centers on leveraging deep learning for feature representation. $\text{Aerial+}$ utilizes an **under-complete denoising autoencoder** (trained on one-hot encoded data) to map the high-dimensional input into a compact, low-dimensional latent space. This encoder-decoder structure forces the model to learn the most prominent, underlying feature associations. Rule extraction is achieved through a controlled "probing" mechanism: a test vector is constructed by deterministically setting the antecedent features ($A$) to probability 1. A forward pass through the trained autoencoder, followed by analyzing the decoder's output probabilities, infers the consequent features ($C$). If the output probability for $C$ exceeds a threshold ($\tau_c$), the rule $A \rightarrow C$ is asserted. This process formally approximates the conditional probability $\mathbb{P}(x_j=c' | x_i=c)$, providing a strong theoretical grounding for the neurosymbolic link.

The method was rigorously evaluated on five real-world UCI ML datasets (Kelly et al., 2023), comparing $\text{Aerial+}$ against seven state-of-the-art baselines. Key results demonstrate that $\text{Aerial+}$ generates a significantly *more concise set of high-quality rules with full data coverage* compared to both exhaustive methods (like FP-Growth) and optimization-based approaches. In terms of efficiency, $\text{Aerial+}$ substantially reduces execution time, particularly as the number of antecedents increases, where exhaustive methods can take hours and yield millions of rules. Furthermore, when these rules are passed to rule-based classifiers (e.g., CBA, BRL, CORELS), $\text{Aerial+}$ maintains or improves accuracy while drastically cutting down the overall classification runtime.

The framework's scalability is also highlighted: training is linear in the number of transactions ($O(n)$), and rule extraction is polynomial in the number of features ($O(k^{a+1})$). $\text{Aerial+}$ is designed to be modular, offering variants for frequent itemset mining and ARM with item constraints, positioning it as a comprehensive neurosymbolic complement to existing ARM literature.
