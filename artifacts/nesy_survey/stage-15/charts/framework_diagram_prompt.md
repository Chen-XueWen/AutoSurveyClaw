# Framework Diagram Prompt

**Paper**: Neurosymbolic AI: A Comprehensive Review of Integration, Reasoning, and Trust

## Image Generation Prompt

Generate a professional academic architecture diagram for Neurosymbolic AI. Style: Flat vector illustration suitable for a top-tier ML conference paper. Background: Very light grey #F8F9FA. Layout: Strict left-to-right data flow pipeline. Visual Elements: Five rounded rectangular nodes connected by thick directional arrows. Node 1: "Input Perception" in muted blue #4477AA. Node 2: "Neural Processing" in teal #44AA99. Node 3: "Symbolic Reasoning" in soft purple #AA3377. Node 4: "Causal Inference" in muted blue #4477AA. Node 5: "Trust Verification" in warm accent #CCBB44. Text: Sans-serif font, white or dark grey for contrast, minimal labels only. Strokes: Thin dark grey borders. Shadows: Subtle drop shadows for depth, no 3D effects. Annotations: Include small side labels on arrows indicating "Feature Extraction", "Logic Derivation", "Verification". Composition: Symmetrical, high information density without clutter. Ensure all hex colors are consistent with the palette. No photorealism, no textures, clean vector lines. The diagram must visually represent the integration of neural fluency with symbolic logic and causal trust. Add a title box at the top: "Neurosymbolic Trust Framework". Arrows should be straight lines with arrowheads. Ensure spacing is uniform and labels are centered within boxes. The final image should look like a high-resolution SVG export ready for publication.

## Usage Instructions

1. Copy the prompt above into an AI image generator (DALL-E 3, Midjourney, Ideogram, etc.)
2. Generate the image at high resolution (2048x1024 or similar landscape)
3. Save as `framework_diagram.png` in the same `charts/` folder
4. Insert into the paper's Method section using:
   - LaTeX: `\includegraphics[width=\textwidth]{charts/framework_diagram.png}`
   - Markdown: `![Framework Overview](charts/framework_diagram.png)`
